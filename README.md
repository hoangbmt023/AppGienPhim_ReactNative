# 🎬 AppXemPhim-UI - Ứng Dụng Xem Phim Trực Tuyến

Ứng dụng mobile xem phim trực tuyến được phát triển bằng **React Native** và **Expo**, cung cấp trải nghiệm xem phim chất lượng cao với đầy đủ tính năng.

## 📚 Tài Liệu

- **[README_PRODUCT.md](./README_PRODUCT.md)** - Hướng dẫn chi tiết về cách hoạt động & kiến trúc ứng dụng
- **[package.json](./package.json)** - Danh sách dependencies

## 🚀 Bắt Đầu Nhanh

```bash
# Cài đặt dependencies
npm install

# Chạy ứng dụng
npm start

# Hoặc chạy trên platform cụ thể
npm run android    # Android
npm run ios        # iOS
npm run web        # Web
```

## ✨ Tính Năng Chính

- ✅ Đăng nhập/Đăng ký tài khoản
- ✅ Khám phá & tìm kiếm phim
- ✅ Xem phim trực tuyến
- ✅ Quản lý gói thành viên
- ✅ Lịch sử xem & giao dịch
- ✅ Quản lý hồ sơ cá nhân

## 📱 Công Nghệ

- React Native 0.79.3
- Expo 53.0.10
- React Navigation
- AsyncStorage
- Axios
